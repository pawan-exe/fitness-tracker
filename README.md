# 🏋️ Fitness Tracker API

Fitness Tracker is a **Spring Boot backend application** that helps users track workouts, manage fitness activities, and receive personalized workout recommendations. The system provides secure authentication and RESTful APIs for fitness tracking.

---

## 🖥️ Tech Stack

- Spring Boot
- Spring Security
- JWT Authentication
- Docker
- Swagger
- Cloud Deployment

---

## 🚀 Key Features

### 🏃 Activity Management
- Save workout activities
- Fetch user activity history
- Track fitness progress

### 🤖 Workout Recommendations
- Generate personalized workout recommendations
- Fetch recommended activities
- Activity-based recommendation system

### 🔐 User Authentication
- User registration
- Secure login system
- JWT-based authentication with Spring Security

### ⚡ RESTful API Structure
- Organized APIs for Activity, Recommendation, and User modules
- Easy integration with frontend or mobile apps

### ☁️ Deployment Ready
- Cloud deployed backend
- Docker support for containerized deployment
